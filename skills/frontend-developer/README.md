# Frontend Developer

A ChatGPT Skill focused on creating polished, production-grade frontend interfaces.

## What this skill does

This skill helps ChatGPT:
- build React components
- design landing pages
- create dashboards
- improve UI styling
- refactor rough interfaces into polished frontend work
